HTML

  • HTML means Hypertext Markup Language.
  • HTML gives content structure and is made up of elements which work together to provide content and basic framework to a website.
  • Elements = entire element from opening to closing tag.
  • Tag = only what apears inside the angle brackets. (< >)
  • Semantic HTML elements are best practice for accessibility features. The name of semantic elements describes its content and the role it plays on the webpage.
  • Child Elements are elements nested inside other elements. Parent Elements are the elements that hold all the "children".
  • head and body are the two most important elements.
  • Always start with the DOCTYPE (not case-senseitive)
  • Elements that are used to divide the content on the page and describe the content to browsers (and cannot be seen on the page) are: Meta-related elements, Content division elements and semantic elements.
  • The head element contains information about the webpage.
  • head contains:
  • lang(Language), title for SEO purposes, style for CSS, bas to define the base URL for a webpage, link to connect external resource to the HTML doc, meta to define meta data (like character sett and viewport), and script to embeds client-side scripts in the HTML document.
  • The body element represents the visible content shown to the user.
  • img elements are self-closing. They require a src and alt text.
